# Olho no Lance TV — Placar Online para OBS

Placar online em tempo real para transmissões no OBS Studio. Controle o placar, cronômetro e cartões diretamente pelo navegador, sem instalar nada.

## Como usar

1. Abra a página de **Controle** no seu celular ou computador.
2. Adicione a página de **Overlay** como fonte do tipo *Browser* no OBS Studio.
3. Use o painel de controle para atualizar placar, cronômetro e cartões — o overlay atualiza automaticamente.

## Funcionalidades

- Placar em tempo real para dois times
- Cronômetro com controles de início, pausa, zerar e ajuste de minutos
- Seleção de período (1º Tempo, 2º Tempo, Intervalo, Prorrogação, Pênaltis, Encerrado)
- Controle de cartões amarelos e vermelhos
- Seleção de times com abreviações personalizáveis
